# 📚 BookBee - Online Book Borrowing Platform

BookBee is an online book borrowing platform where users can explore, borrow, and manage books easily. The platform provides a modern and user-friendly interface for discovering books by category, viewing details, and managing user profiles.

---

## 🌐 Live Website

🔗 https://bookbee-online-platform.vercel.app/

---

## 🎯 Project Purpose

The purpose of BookBee is to create a simple and interactive online library system that allows users to:

- Browse books easily
- View detailed book information
- Borrow books online
- Manage their profiles
- Explore books by categories

This project was built to practice and demonstrate modern full-stack web development skills using Next.js, MongoDB, and authentication systems.

---

## ✨ Key Features

- 🔐 User Authentication (Login / Register / Logout)
- 📚 Browse all books
- 🔎 Search books by title
- 🏷️ Filter books by category
- 📖 View detailed book information
- 👤 User profile management
- 🎨 Responsive modern UI
- 🐝 Custom BookBee branding and theme
- ⚡ Dynamic routing with Next.js
- 🔒 Protected routes
- 📱 Mobile responsive design
